NumberInput Component - Range Start/End Enhancement

The List Manager now features an improved user experience for setting caddie range values. Users can now easily adjust values using increment/decrement buttons OR type directly for precise control.

Before (Number Input Only)

Users had to manually type or use browser's small arrows

After (NumberInput with Buttons)

Easy increment/decrement + direct typing

New Features

  • Large, easy-to-click increment/decrement buttons
  • Free text input for direct number entry
  • Min/max value enforcement (Range Start ≤ Range End)
  • Disabled state for buttons at boundaries
  • Smooth animations and hover effects
  • Fully keyboard accessible
  • Matches the app's design system
